# كرم و ميرا 💕 — ذكرياتنا

موقع صغير ولطيف لحفظ ذكرياتك مع ميرا.

## كيف تشغّل الموقع

1. افتح المجلد `mieraye` في جهازك.
2. افتح الملف `index.html` بمزدوج النقر (أو اسحبه إلى المتصفح).

أو من الطرفية:

```bash
cd mieraye
start index.html
```

## المميزات

- **إضافة ذكريات**: عنوان، تاريخ، وصف، وصورة (رابط اختياري).
- **عرض الذكريات** على شكل بطاقات مرتبة من الأحدث للأقدم.
- **حذف ذكرى** من زر "حذف" في كل بطاقة.
- **التخزين محلياً** في المتصفح (localStorage) — الذكريات تبقى على نفس الجهاز والمتصفح.

## الملفات

- `index.html` — الصفحة الرئيسية
- `styles.css` — التصميم (ألوان pastel، خطوط عربية، تأثيرات لطيفة)
- `app.js` — منطق إضافة/عرض/حذف الذكريات
